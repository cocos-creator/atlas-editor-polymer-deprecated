基础操作：
- 添加图片：拖放图片文件到画布上。Webkit内核浏览器或离线版支持拖放文件夹。
- 删除图片：左键选中图片，按Delete(PC)或Ctrl+Backspace(MAC)进行删除。
- 鼠标：滑动滚轮缩放画布，中键或右键拖动平移画布。
